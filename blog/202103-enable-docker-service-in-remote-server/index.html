雲端主機啟動 Docker 服務

介紹如何在雲端主機啟用 docker 服務

2021/03/13

Ray

新開主機啟用 docker 與 docker-compose 服務的操作步驟

安裝 Docker

sudo apt-get install docker.io
sudo apt install docker.io

將使用者加入群組

此操作可以讓使用者不需要 sudo 就執行 docker 指令，為非必要步驟

sudo usermod -a -G docker {ec2-user}

ref: Ubuntu Linux 安裝 Docker 步驟與使用教學

也有人直接將使用者加入 sudoer 群組中，有一樣的結果

安裝 docker-compose

sudo curl -L "https://github.com/docker/compose/releases/download/1.29.1/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose

ref: Install Docker Compose

如此就可以在新開的雲端主機進行 Docker 容器的操作
