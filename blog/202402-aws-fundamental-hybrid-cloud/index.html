混合雲快速一覽

2024/02/13

Ray

混合雲端是一種IT 基礎設施設計，它可將公司的內部IT 資源與第三方雲端供應商的基礎設施和服務整合在一起
AWS 混合雲端服務可在您需要的任何地方提供一致的AWS 體驗– 從雲端、內部部署到邊緣
藉助混合雲端，您可以儲存資料並跨多個環境執行應用程式

第一代混合雲

  1. Direct Connect ：建立實體機房和 AWS 的私有專用網路連線
  2. Storage Gateway ：在本地模擬一個網路檔案系統(Network File System, NFS)
    使用者上傳檔案到既有機房時會同步上傳到雲端上(S3, EBS, Glacier)
  3. VM Import/Export ：遷移虛擬機器，把實體伺服器內的 VM 轉換到雲端(EC2)

第二代混合雲的崛起

保留架構彈性，引入容器，增強乘載能力

  1. Route53 ：雲端 DNS 將流量平均分散到本地機房和雲端機房
  2. Direct Connect ：雲端和本地雙邊資料同步用的私有專線
  3. FSx ：高速資料同步，縮短同步延遲時間
  4. ECS Anywhere ：AWS 的本地容器代管服務，可以用 AWS 的介面管理本地應用的容器
  5. VM Import/Export ：匯入/匯出 EC2 映像檔，自由遷移
