Ngrok 讓 localhost 可以被外部連線

Ngrok 開發測試 webhook 的利器

2021/02/14

Ray

ngrok 做為一個轉發的伺服器，他可以把外界的請求轉發到你指定的 Port
使用的背景原理是連接到 ngrok 雲端伺服器將你本機指定的地址公開
再將由 ngrok 一串公開的網址來存取內容
他的優點是快速而且還提供了 https 的服務讓你使用上更安全，甚至你還可以設置密碼保護
